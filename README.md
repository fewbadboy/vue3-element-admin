# vue3-element-admin

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Customize configuration
```
npm install --registry=https://registry.npm.taobao.org
yarn add --dev
```
